Notable projects

For more information on my open source projects, consult my projects.

cargo : : generic R data containers, schemas & prototypes 2020 - current
Open source: https://github.com/jeanmathieupotvin/cargo.

A package that implements generic low-level S4 containers for data, data schemas and data prototypes. These containers are intended for production-grade applications. They bring formality and increased reliability at a negligible cost in resources. They are easily customizable through a user-friendly API and easily extendable to other S4 super-classes.

fs : : flood simulators 2018 - 2020
Closed source. © The Co-operators General Insurance Company

A tool for plausible large-scale simulations of dependent flood losses stemming from fluvial and/or coastal flood events. It is based on a collection of multiple sub-models organized in a hierarchical way into so-called pipelines. The input data flows through those pipelines to generate stochastic simulations of plausible flood losses. An internal package serves as a user-friendly interface to these simulator.

qch : : Quebec's hydrometry 2019
Closed source. © The Co-operators General Insurance Company

A modern API to get river flows, river levels and gauging stations metadata easily from the web. This package provides an easy-to-use and fully automated API to fetch and format data stemming from the usual CEHQ, MDDELCC and MELCC websites. Standardized databases are created from these sources.

Future open-source project. I plan to rebuild this package from scratch in 2021. It will be open source and available to everyone. It is an essential tool for anyone trying to do serious flood modelling and/or flood risk assessment.
