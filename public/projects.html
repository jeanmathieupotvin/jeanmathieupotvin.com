cargo :: generic R data containers, schemas & prototypes

General low-level S4 containers for data, data schemas and prototypes. These containers brings formality and increased reliability at a negligible cost in resources, with a focus on production-grade applications. They are easily customizable through a user-friendly API and easily extendable to other S4 super-classes. This project leverages the well-known features of the data.table package to work by reference whenever it is possible for faster results.
